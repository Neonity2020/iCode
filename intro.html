iCode

在桌面驱动 Codex
完成你的编码任务

基于 Electron + React 的本地桌面工作台，通过真实 Codex 事件流管理任务会话、模型切换、审批确认与状态持久化，不依赖模拟数据。

快速开始

前提：已安装 pnpm，并已安装登录 Codex CLI。

# 安装依赖
pnpm install

# 启动完整开发环境（Vite + Electron）
pnpm run dev

# 仅前端开发服务
pnpm run dev:web

使用流程

几步即可开始你的第一个任务。

  1. 启动应用后，先选择一个工作区目录。
  2. 点击「新任务」创建会话。
  3. 输入你希望 Codex 完成的工作。
  4. 当 Codex 请求权限时，按需允许或拒绝。
  5. 如需停止当前任务，点击停止按钮。

工作原理

界面中的一切都来自真实的 Codex 事件流。

iCode 主进程会启动 codex app-server --listen stdio://，并通过 preload 暴露最小化的 IPC 接口给渲染进程。消息、活动、审批和错误状态全部来自真实事件流；会话状态、工作区、模型选择与侧栏状态会保存在本地，重启后自动恢复最近的任务上下文。
